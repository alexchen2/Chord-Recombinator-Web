Chord Recombinator

Select a group of notes through various input methods and generate various chords containing them using our chord recombinator tool.

Use a Virtual Piano Keyboard

Select the keys on an interactive piano keyboard and discover chords using a combination of notes.

Use your Device's Microphone

Simply record yourself playing the notes using your instrument of choice—we’ll detect the notes for you and take it from there.

Other Methods of Input

Use your mouse to place notes on a music staff and build a chord, then use our tool to identify it. Staff Builder

If you know the note names, you can manually enter them in text fields. Text Input

More input options will be added in the future, such as visual input for instruments besides the piano.
